FREE Free 플랜의 제한 (3가지 암기)
  • 메시지 보관 90일 제한 — 이후 메시지는 검색 불가
  • 앱 연동 최대 10개 제한
  • 외부 협업(Slack Connect)은 1:1 DM만 가능, 채널 초대 불가
PRO Pro부터 해금되는 것
  • 메시지 & 파일 무제한 보관
  • 앱 연동 무제한
  • 외부 조직과 채널 협업 가능 (최대 250개 외부 조직)
  • 액세스 로그 및 기본 SSO 지원
BUSINESS+ Business+부터 해금되는 것
  • SAML 기반 SSO — 기업 IdP(Okta, Azure AD 등) 연동
  • SCIM 사용자 자동 프로비저닝/디프로비저닝
  • 기본 DLP(데이터 손실 방지)
  • EMM은 여기서 안 됨 — Enterprise Grid 전용
ENTERPRISE GRID Enterprise Grid만 되는 것 시험 빈출
  • EMM (Enterprise Mobility Management) — 모바일 기기 접근 제한
  • eDiscovery / 법적 자료 보존 (Legal Hold)
  • 정보 장벽 (Information Barriers) — 부서 간 소통 차단
  • HIPAA 준수 지원 — 의료·금융 규정
  • 다중 워크스페이스 통합 관리 — 여러 WS를 하나의 Grid로
  • 고급 DLP 및 엔터프라이즈 검색
